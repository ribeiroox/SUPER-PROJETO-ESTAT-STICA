# SUPER-PROJETO-ESTAT-STICA
# Amazon Products — Estatística e Probabilidade

Projeto final da disciplina de Estatística e Probabilidade.
Análise exploratória, probabilidade bayesiana e classificação de produtos da Amazon India.

## Arquivos

| Arquivo | Descrição |
|---|---|
| Amazon_completo.ipynb | Notebook completo: limpeza, EDA, Bayes e algoritmos |
| amazon.csv | Dataset original (Amazon India) |
| dashboard_amazon.html | Dashboard interativo — abre no navegador |
| Relatorio_Tecnico_Completo.docx | Relatório técnico do projeto |

## Como executar o notebook

1. Instale as dependências: `pip install pandas numpy scikit-learn matplotlib seaborn`
2. Abra o arquivo `Amazon_completo.ipynb` no Jupyter Notebook ou Google Colab
3. Execute as células em ordem

## Como usar o dashboard

Abra o arquivo `dashboard_amazon.html` direto no navegador (Chrome ou Firefox).
Não precisa instalar nada.

## Dataset

Amazon Products Dataset — Amazon India
Fonte: [Kaggle](https://www.kaggle.com/datasets/lokeshparab/amazon-products-dataset)
